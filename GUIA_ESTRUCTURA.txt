# 📂 Guía Maestra de Estructura - SITIO WEB ELECTRÓNICO

Este sitio ha sido convertido de WordPress a HTML estático. Aquí te explico qué es cada cosa para que no te pierdas:

## 🏠 Raíz del Proyecto (/)
- **index.html**: La página principal de tu sitio. Contiene todo el diseño de inicio.
- **dashboard.html**: Tu nuevo panel privado para ver Google Analytics.
- **GUIA_ESTRUCTURA.txt**: Este archivo que estás leyendo.
- **Otras carpetas (shop, checkout, etc.)**: Contienen las subpáginas de tu sitio. Cada una tiene su propio `index.html`.

## 🖼️ Recursos y Activos
- **wp-content/**: ¡Aquí vive casi todo lo visual! 
  - `uploads/`: Todas tus imágenes, fotos y videos.
  - `themes/`: Los estilos (CSS) que hacen que el sitio se vea bonito (Astra).
  - `plugins/`: Funciones extra como los bloques de Spectra que usamos.
- **wp-includes/**: Archivos base necesarios para que cosas como los emojis o scripts básicos funcionen.

---
💡 **TIP DE HACKER:** Todas las rutas ahora son relativas (empiezan con `./`), lo que permite que el sitio funcione en cualquier carpeta de GitHub Pages.
